# 박스권 스캐너 컨트롤룸

코스피 전종목을 스캔해서 박스권 종목만 자동으로 추려주는 도구입니다.

🔗 **배포 주소**: https://appapppy-arrvupjezplekjddatmdmg.streamlit.app

---

## 📁 파일 구조

```
box-range-scanner/
├─ streamlit_app.py       ← UI 메인
├─ box_range_scanner.py   ← 스캔 로직 모듈
├─ requirements.txt       ← 설치 목록
└─ README.md              ← 이 파일
```

---

## 🔍 스캔 모드

### ⚡ 빠른 스캔 (후보군)
- 거래량 상위 종목 기준 (사이드바에서 수량 설정)
- 빠르게 결과 확인 가능

### 🔍 전체 스캔 (코스피 전종목)
- 코스피 전체 종목(약 900~1,000개) 대상
- 박스권 조건(60점 이상) 만족 종목만 결과에 표시
- 진행바로 실시간 진행 확인 가능
- 소요 시간: 수 분

---

## 📊 결과 화면

| 컬럼 | 설명 |
|------|------|
| 종목코드 | KRX 종목코드 |
| 종목명 | 종목 이름 |
| 점수 | 박스권 점수 (100점 최고) |
| 거래량 | 최근 거래량 |
| 이유 | 감점 사유 또는 "박스권 안정" |
| 돌파신호 | 🟢 상단돌파임박 / 🟡 박스권상단 / ⚪ 박스권중립 / 🔴 하단이탈임박 |

결과 요약 예시:
```
코스피 950개 중 42개 박스권 후보 발견 (60점 이상)
```

---

## 🔖 점수 기준

| 점수 | 의미 |
|------|------|
| 100점 | 박스권 가능성 매우 높음 |
| 70점 | 보통 |
| 40점 이하 | 추세 진행 중 |

감점 기준:
- 변동폭 과다 (고가-저가 범위 30% 초과) → -30점
- 변동성 과다 (일일 변동성 5% 초과) → -30점
- 추세 존재 (이동평균 기울기 급함) → -30점

## ⚙️ 로컬 실행 방법

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

---

## ❗ 오류 대응

| 증상 | 원인 | 해결 |
|------|------|------|
| `결과 없음` | 주말/공휴일 또는 장 마감 전 | fallback 정상 동작, 평일 장 마감 후 재실행 |
| `종목 목록을 가져올 수 없습니다` | pykrx 네트워크 오류 | 잠시 후 재시도 |
| `ModuleNotFoundError` | 패키지 미설치 | `pip install -r requirements.txt` 재실행 |
